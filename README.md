Public Key Cryptography course assignments

Ruby version: 2.5.2
Rails version: 5.1.4

# Run locally

In order to run this project:
* clone it using
`git clone https://github.com/rusucosmin/cryptography`
* navigate to the project
`cd cryptography`
* run the local instance
`rails server`
